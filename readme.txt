Blackjack.py
Version 1
Author Joseph Cheatham

This project is based on something I made as a mini game for a text rpg in C++, converted to python.
It currently uses a simple loop structure to manage the drawing and then nested if statements to compare
hands. The player is competing against the "dealer" and can win or lose to the dealer, but not every game will
have a winner (i.e both the player's and dealer's hand exceeds 21). I'm probably going to add actual card names
with a parallel list structure eventually.

How to play:
You are competing against the "dealer" for a better hand. But exceeding a hand value of 21
will automatically lose the game. Both draw initially. You will choose whether or not to draw
with "yes" or "no" for every subsequent round. This continues until the player enters no or
their hand exceeds 21 (busts).

Good luck and have fun!
